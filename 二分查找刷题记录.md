**二分查找**

题解：
排序好的数组，每次取中间的数进行比较，如果相等的话，直接返回该位置，如果小于数组边界左移到中间，大于的话，数组右移到中间。
~~~
class Solution(object):
    def search(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: int
        """
        right = 0
        left = len(nums) - 1
        while(right <= left):
            #print(str(right) + '----'+str(left))
            mid = (left + right)/2 
            if nums[mid] == target:
                return mid
            elif nums[mid]<target:
                right = mid + 1
            else:
                left = mid - 1
        return -1
~~~
