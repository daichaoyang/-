
**丢失的数字**
题解：
直接使用数学公式即可
(（1+n）* n)/2 - sum

~~~
class Solution(object):
    def missingNumber(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        n = len(nums)
        sum_nums = sum(nums)
        return (1 + n)*n/2 - sum_nums
~~~

**翻转链表**

题解：直接将节点的next指向其前一个节点即可，注意首节点

~~~
直接在原链表上翻转
class Solution(object):
    def reverseList(self, head):
        """
        :type head: ListNode
        :rtype: ListNode
        """
        p_head = head
        if head == None:
            return head
        p_head = head
        left_node = head
        while(p_head != None):
            tmp = p_head.next 
            p_head.next = left_node
            left_node = p_head
            p_head = tmp
            if p_head == None:
                break
        head.next = None
        return left_node
	
	使用数组存储链表，然后从尾部开始反连接

        :rtype: ListNode
        """
        p_head = head
        if p_head == None:
            return p_head
        linklist = []
        while (p_head != None):
            linklist.append(p_head)
            p_head = p_head.next
        for i in range(len(linklist)-1, 0, -1):
            linklist[i].next = linklist[i-1]
        linklist[0].next = None
        return linklist[len(linklist)-1]
